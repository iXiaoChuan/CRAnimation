## 使用：

导入`HZLaunchView.h`

```
//初始化
HZLaunchView *launchView=[[HZLaunchView alloc] initWithIconImage:[UIImage imageNamed:kTwitterImageName] backgroundColor:color];

//指定动画时间,启用动画
[launchView startAnimationWithDuration:3];
```
